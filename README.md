# PDF Multi-Column Parser (PyMuPDF)

This project parses **text-based multi-column PDFs** and classifies content into:
- TITLE
- HEADING
- SUBHEADING
- BODY

## Requirements
- Python 3.9+
- PyMuPDF

## Installation
```bash
pip install pymupdf
